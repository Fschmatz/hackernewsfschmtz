# HackerNews Fschmtz

Application used to view the website https://news.ycombinator.com/news

```
-> Top Stories
-> New Stories
-> Best Stories
-> Ask HN
-> Show HN
```


Links and comments are opened externally via the browser.


Created with Android in mind, it has not been tested for IOS.
