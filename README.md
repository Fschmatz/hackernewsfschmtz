# HackerNews Fschmtz

Flutter application using the API from <a href="https://news.ycombinator.com/news">Hacker News</a>

```
Pages:
-> Top Stories
-> New Stories
-> Best Stories
-> Ask HN
-> Show HN
```


Links and comments are opened externally via the browser.
